## Sommes remarquables

* Somme des premiers entiers $\sum_{k = 1}^{n}{k = \frac{n \cdot (n + 1)}{2}}$
* Somme des q puissances $\sum_{k = 0}^{n}{q^{n} = \frac{1 - q^{n + 1}}{1 - q}}$ 
* Nbre d'éléments $\sum_{k = 1}^{n}{} = (n - k + 1)$
##  Les suites
 
* Suite arithmétique :
    * $u_{n + 1} = u_{n + 1} + a$
    * $u_{n} = u_{n_0} + a \cdot (n - n_0)$
* Suite géométrique :
    * $u_{n + 1} = q \cdot u_n$
    * $u_{n} = u_{n_0} \cdot q^{n - n_0}$
